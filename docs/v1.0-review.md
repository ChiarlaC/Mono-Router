## 二、中间的卡点与解决

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 滚动条不显示 | Gradio Textbox 默认不限制行数 | 添加 `max_lines=30` 参数 |
| Markdown 符号残留 | 提示词未明确禁止格式符号 | 更新提示词要求纯文本/HTML输出 |
| 复制到公众号无样式 | 纯文本无法保留格式 | 改为 HTML 输出 + `gr.Code` 组件 |
| Git 推送被拒绝 | 远程仓库已有内容 | 强制覆盖推送（用户确认） |
